# OnePage Starter

Minimales React-Grundgeruest fuer eine OnePage, die wir Schritt fuer Schritt gemeinsam aufbauen koennen.

## Enthalten

- React + Vite als schlanke Basis
- vorbereitete GSAP-Dependency fuer spaetere Animationen
- globale Style-Struktur mit Reset, Tokens und App-weiten Styles
- neutrale Seitenhuelle ohne inhaltliche Platzhalter

## Start

Sobald Node/npm lokal verfuegbar sind:

```bash
npm install
npm run dev
```

## Struktur

```text
src/
  components/
  lib/
  styles/
  App.jsx
  main.jsx
```
